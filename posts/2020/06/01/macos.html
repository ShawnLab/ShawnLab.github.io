# 使用 Tab 键快速切换对话框按钮

macOS 把对话框按钮分为默认选项（蓝色背景按钮）和键盘焦点（蓝色方框包裹按钮）两种。其中，默认选项通过回车键来确认选中，是由对话框的属性决定的，不能更改；键盘焦点可以通过 Tab 键来移动，并通过空格键来确认选中。

  • 进入系统偏好设置 -> 键盘 -> 快捷键 -> 在左下角选中使用键盘导航在控制间移动焦点

# 在特定的 App 里显示 fn 功能键

在使用集成开发和文本编辑器时，经常会使用传统的 F1-F12 功能键。但使用时要先按下 fn 键，降低了效率，此时可以设置特定 App 在触控栏中显示功能键而非 App 控制。

  • 进入系统偏好设置 -> 键盘 -> 快捷键 -> 在左侧选择fn 功能键，将你需要使用功能键的 App 添加到右侧列表，然后你就能在这些 App 里使用功能键了。
